## 🚀 Featured Projects

### 🏥 [MediGuide](https://github.com/ramzimarir/MediGuide)
> AI-powered medical prescription assistant

GraphRAG pipeline combining **Qdrant semantic search** and **Neo4j knowledge graph**
(8,000+ medicines) to generate evidence-based medication recommendations from
clinical consultation notes — with speech transcription, NER extraction,
and contraindication checking.

`FastAPI` `Neo4j` `Qdrant` `GraphRAG` `Groq` `EdenAI` `Docker`

---

### 🦾 [Gait Event Detection](https://github.com/ramzimarir/gait-event-detection)
> Deep learning for clinical gait analysis · LISSI Lab, UPEC

Benchmarking **CNN, LSTM, and TCN** architectures for automatic detection
of gait events (Initial Contact / Toe-Off) from foot-mounted IMU signals,
evaluated under Leave-One-Subject-Out cross-validation across 10 subjects.

Best result: **LSTM — 91.4% accuracy @ 50ms tolerance**

`PyTorch` `LSTM` `TCN` `CNN` `IMU signals` `Research`
